# Vinland
> Una aplicación de mapeo y exploración para uso recreativo

Este README está disponible en: [Français](README.md) , [English](README-EN.md) , [Latin](README-LAT.md)

## Uso

¿Te gustaría incorporar elementos de juego de rol en tu experiencia de juego pero no tienes una herramienta adecuada? Vinland te permite crear campañas con mapas personalizables para tus sesiones de juego de rol. Simplemente crea una cuenta de Maestro de Juego (GM), establece tus campañas junto con sus mapas únicos e invita a tus amigos a unirse a la aventura.

## Instalación

1. Descarga la carpeta VinlandSol.
2. Abre el archivo VinlandSol.sln utilizando Visual Studio 2022.
3. Inicia el proyecto VinlandSol.
